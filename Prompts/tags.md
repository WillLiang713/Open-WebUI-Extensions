### 任务:
  根据聊天记录生成 1-3 个宽泛主题标签，以及 1-3 个更具体的子主题标签。

### 要求:
  - 先给出高层领域标签（如：科学、技术、哲学、艺术、政治、商业、健康、体育、娱乐、教育）
  - 若对话中有明显子领域，可补充更具体的标签
  - 如果内容过短（少于 3 条消息）或过于分散，仅输出 ["通用"]
  - 使用对话的主要语言；中文优先
  - 宁可准确清晰，不要过度细分

### 输出:
  JSON 格式: { "tags": ["标签1", "标签2", "标签3"] }
